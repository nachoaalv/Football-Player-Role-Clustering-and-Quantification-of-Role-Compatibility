# Synergy Matrices

This folder contains CSV files representing synergy matrices between player roles.

### Files
- `ball_recoveries_per90_matrix.csv`
- `carries_per90_matrix.csv`
- `defensive_actions_per90_matrix.csv`
- `minutes matrix.csv`
- `pass matrix`
- `pressures_per90_matrix.csv`
- `shotassist matrix`
- `shots_dribbles_per90_matrix.csv`
- `win percentage matrix.csv`
- `xgagainst matrix.csv`
- `xgfor matrix.csv`
  
### Purpose
Synergy matrices quantify how role combinations affect team and individual performance metrics. 
They have a synergy score for each possible pair of roles on the pitch.
They are used by the lineup optimiser to build balanced XIs.
For more information on the matrices check `dissertation/Football Player Role Clustering and Quantification of Role Compatibility`
