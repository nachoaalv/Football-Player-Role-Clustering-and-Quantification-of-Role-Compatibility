# Notebooks

This folder contains Jupyter notebooks used for analysis and experimentation.

### Main notebooks
- `final_lineup_optimiser.ipynb` — Builds lineups using player roles and synergy matrices.
- `synergy_matrices_final.ipynb` — Generates synergy matrices from processed player data.

### Position clustering notebooks
Located in `positions/`:
- `goalkeepers.ipynb`
- `centerbacks.ipynb`
- `fullbacks.ipynb`
- `wingers.ipynb`
- `strikers.ipynb`
- `midfielders.ipynb` (all midfielders together)
- `midfielders_split.ipynb` (final version, splits midfielders into sub-groups)

These notebooks cluster players into roles using k-means and dimensionality reduction.
