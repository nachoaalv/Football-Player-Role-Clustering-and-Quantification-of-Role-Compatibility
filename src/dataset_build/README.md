# Dataset Build Scripts

This folder contains notebooks used to build player datasets for each league.

### Files
- `dataset build final-bundesliga.ipynb`
- `dataset build final-laliga.ipynb`
- `dataset build final-ligue1.ipynb`
- `dataset build final-prem.ipynb`
- `dataset build final-serieA.ipynb`

Each notebook processes StatsBomb event data into season-long player statistics,  
which are then used for clustering and role assignment.
